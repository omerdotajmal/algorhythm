# Design & Analysis of Algorithms — Code + Visualizations

This repository contains selected **code examples** and **interactive visualizations** used in the **Design & Analysis of Algorithms (DAA)** course.  
The goal is to support learning through small, focused implementations and step-by-step algorithm traces.

## What’s inside

### Brute Force (current)
- **Naive String Matching** — tries every shift and compares characters.
- **Closest Pair (2D) — Brute Force** — checks every pair of points to find the minimum distance.
- **Subset Generation (Bitmasking)** — enumerates all masks from `0..(2^n - 1)` and decodes subsets.

> More techniques (Divide & Conquer, Greedy, Dynamic Programming, Graph Algorithms, etc.) will be added gradually.
